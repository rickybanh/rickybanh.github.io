Pyramid Blending

Computational Photography

Pyramid Blending is an image blending technique that utilizes image pyramids to create smoother blends between two images. Image pyramids are a concept in which an image is repeatedly reduced to create several layers of smaller images that have less high-frequency information. Each reduction leads to a smaller image with increased smoothing but decreased resolution.
